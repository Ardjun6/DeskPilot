## [1.0.0] - 2025-01-24

### üéâ Initial Release

#### Features
- **Actions System**
  - Create and run custom automations
  - Support for multiple step types
  - Global hotkey triggers
  - Favorite actions

- **Visual Workflow Builder**
  - Drag-and-drop interface
  - No coding required
  - Step parameter editing
  - Save as action

- **Action Recorder**
  - Record mouse clicks
  - Record keyboard presses
  - Auto-detect delays
  - Edit before saving

- **Clipboard Manager**
  - Automatic history tracking
  - Search functionality
  - Pin important items
  - Quick paste (Ctrl+Shift+V)

- **Mouse Jiggler**
  - 5 movement patterns
  - Caffeine Mode
  - Scheduled operation
  - Statistics tracking

- **Text Templates**
  - Jinja2 variable support
  - Hotkey triggers
  - Quick copy to clipboard

- **System Tray**
  - Minimize to tray
  - Quick actions menu
  - Jiggle toggle
  - Clipboard access

- **Quick Actions Widget**
  - Floating always-on-top panel
  - Favorite actions
  - Draggable

- **Command Palette**
  - Quick action search
  - Keyboard navigation
  - Recent actions

- **Themes**
  - Productivity (blue)
  - Calm (green)
  - Contrast (high contrast)
  - Playful (purple)
  - Glass (dark modern)

- **UI Polish**
  - Animated splash screen
  - Progress indicators
  - Clean card design
  - Responsive layout
